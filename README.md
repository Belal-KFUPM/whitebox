# Whitebox Demo

Frontend demo for Whitebox Analytics, an explainable industrial risk intelligence platform.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```

The production build is generated in `dist/`.

## Deploy

For Vercel or Netlify:

- Build command: `npm run build`
- Output directory: `dist`
